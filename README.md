## Test
Le but de ce tp est de construire une interface web pour notre application 
développé en Java, nous avons utilisé le tp 4 déja réalisé
Nous avons généré une interface par Yeoman en exécutant la commande Yo
afin de récupérer une personne.
 
Nous avons deux parties principales :
## Le controlleur :
  qui permet de récupérer une personne en utilisant le service $http, 
  le meme fonctionnement que le tp4, il suffit de lancer l'url suivant:
  http://localhost:9000/rest/person 
## La vue :
   Nous avons crée un simple formulaire avec deux attributs; nom et prenom pour    récupérer une personne.
